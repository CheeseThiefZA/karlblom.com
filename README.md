# karlblom.com

Personal website for Karl Blom - Dad, Lawyer, Tech Nerd.

## Hosted on GitHub Pages

This site is hosted for free using [GitHub Pages](https://pages.github.com/).

## Setup Notes

### Contact Form
The contact form uses [Formspree](https://formspree.io/) for form handling. To set up:
1. Create a free account at formspree.io
2. Create a new form and get your form ID
3. Replace `YOUR_FORM_ID` in index.html with your actual form ID

### Custom Domain
The site is configured to use `karlblom.com` via the CNAME file.
